制度改正・対象ファンドの追加・除外があれば `lib/funds.ts` を更新する。

### annualReturns（年次リターン）
毎年末に当該年のリターンを追記する。Stooqからの自動取得が失敗した場合のフォールバック値として機能する。
```
// 例: 2026年末に追記
2026: 0.XXX,
```

## 更新不要（固定情報）

以下は銘柄の本質的な特性であり、ファンド自体が廃止・改名されない限り変更不要。

- `nickname` / `formalName` / `catchCopy`
- `forWhom` / `volatility` / `expectedHorizon` / `beginnerScore`
- `features`（定性的な特徴タグ）
- `pros` / `cons`（企業名ではなく特性を説明しているため）

---

## FANG+ 専用チェック（年1回）

NYSE FANG+指数は年次で構成銘柄を入替することがある。
ただし `lib/funds.ts` の説明文は企業名に依存しない表現に統一済みのため、**通常は変更不要**。

確認事項:
- `description` / `catchCopy` / `pros` に特定企業名が混入していないか目視確認
- 混入していた場合は抽象表現に修正する

---

## SCHD 専用チェック（2〜3年に1回）

`cons` に「実績が浅い」等の時限的表現が混入していないか確認。
現在は抽象表現に統一済みのため不要。
